# Prompt task app

Build a PWA as a task manager. The PWA should work on iOS/android/windows/macOS and the backend should be a CalDAV server (or multiple) where the user logs into and sync tasks.
The app should support multiple tasks lists, which can be organized in folders and recursive subfolders.
The app should support tags (identified with a hashtag (#) and nested tags. The app should also support smart search filters, using (nested) tags, folders, and properties of the task (e.g. start/end date). These filters should be stored in caldav in a hidden tasklist. The name of the task can be the name of the smart list and the text note should describe the filter.
If possible in a PWA, it should provide notifications as well.
